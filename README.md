# English Dictionary - 英语单词学习应用

一个基于艾宾浩斯遗忘曲线的英语单词记忆应用，帮助用户高效学习和复习英语单词。

## 功能特性

### Dashboard 仪表盘
- **每日复习目标** - 可视化展示今日复习进度
- **签到系统** - 连续签到获取徽章奖励
- **学习热力图** - GitHub 风格的年度学习记录
- **每日名言** - 随机展示英语名句激励学习

### 单词管理
- **添加单词** - 支持添加新单词及其释义
- **单词列表** - 管理已添加的所有单词
- **复习单词** - 基于遗忘曲线智能安排复习

### 名言管理
- 管理和浏览英语名言警句

## 快速开始

### 环境要求
- Node.js >= 18
- pnpm (推荐) 或 npm

### 安装依赖

```bash
pnpm install
```

### 配置环境变量

复制环境变量示例文件：

```bash
cp env-example .env
```

根据实际情况修改 `.env` 文件中的配置。

### 启动开发服务器

```bash
pnpm dev
```

应用将在 http://localhost:5173 启动。

### 构建生产版本

```bash
pnpm build
```

### 预览生产版本

```bash
pnpm preview
```
